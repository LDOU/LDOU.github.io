Posted on 2020-08-20 Edited on 2020-08-26

原始类型:

  • Boolean
  • Null
  • Undefined
  • Number
  • BigInt
  • String
  • Symbol
    引用类型：
  • Object

注意：

  • js只有一种数字类型，不管是符号值还是NaN都是Number类型
  • 对象、数组、Data、正则都是Object [] instanceof Array //true
    new Date() instanceof Date //true
    null instanceof null //报错 Object.prototype.toString.call(‘’) //[object String]
    Object.prototype.toString.call(1) //[object Number]
    Object.prototype.toString.call(undefined) //[object undefined]
    Object.prototype.toString.call(new Function()) //[object Function]
    Object.prototype.toString.call([]) //[object Array] 1 + ‘1’ //‘11’
    2 * ‘2’ // 4
    [1, 2] + [2, 1] //‘1,22,1’
    // [1, 2].toString() -> ‘1,2’
    // [2, 1].toString() -> ‘2,1’
    // ‘1,2’ + ‘2,1’ = ‘1,22,1’

‘a’ + + ‘b’ //‘aNaN’
// + ‘b’ -> NaN



x == y   返回  true  或者  false

+ x、y类型相同
  x为undefined  => true
  x为null  => true
  x为number  
       x为NaN   => true
       x、y值相等   => true
       x为 +0 y为 -0   => true
  x为String，y跟他完全相同字符序列   => true
  x为Boolean，同时为true或者false   => true
  x、y引用同一对象   => true
+ x为null，y为undefined   => true
+ x为Number，y为String 返回比较x == toNumber(y)的结果
+ x为Boolean，返回比较toNumber(x) == y的结果
+ x为String或Number，且y为Object，返回比较 x == y的结果
